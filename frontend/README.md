# AgentERP Frontend

This is the React frontend for AgentERP, a chat-based interface for AI-powered ERP operations.

## Quick Start

See the main [README.md](../README.md) in the project root for complete setup instructions.

### Prerequisites

- Node.js 16+
- Yarn

### Installation

```bash
cd frontend
yarn install
yarn start
```

The frontend will be available at http://localhost:3000

## Features

- **Chat Interface**: Natural language interaction with ERP systems
- **Real-time Updates**: Live status monitoring and notifications
- **Visual Widgets**: Interactive ERP data visualization
- **Session Management**: Persistent chat sessions
- **Responsive Design**: Modern UI with Tailwind CSS

## Tech Stack

- React 19
- React Router DOM
- Tailwind CSS
- Radix UI Components
- Axios for API calls
- Lucide React icons

## Development

### Available Scripts

- `yarn start` - Start development server
- `yarn build` - Build for production
- `yarn test` - Run tests

### Project Structure

```
src/
├── components/     # Reusable UI components
├── hooks/         # Custom React hooks
├── lib/           # Utilities and API clients
├── App.js         # Main app component
└── index.js       # Entry point
```
